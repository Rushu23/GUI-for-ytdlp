# GUI-for-ytdlp
This is JUST Graphical interface for yt-dlp that make it easier to use (made for newest LINUX MINT)
How this works? So normally if you want to use yt-dlp you need to type some stuff in console which is unpleasant.
Thats why i came with this gui that will do it for u. Just enter what u want to download, then set some parameters and enjoy!
Of course you need to get real yt-dlp first.
It will work anywhere where yt-dlp works.
